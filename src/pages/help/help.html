IsThisJobGreen? Help

Guidance on setup, badge meanings, caveats, and data sources.

How to use

  1. Install and enable the extension (or embed the widget on your site).
  2. Set your home postcode, commute mode, and office days per week.
  3. Browse job listings or open a role to see the commute estimate badge.

Badge states

  • Numeric value (e.g. 120 kgCO2e/yr) is an estimated annual commute.
  • 0 kgCO2e/yr indicates a remote or work-from-home role.
  • No data / Unknown appears when a location is missing or too broad.
  • Set postcode prompts you to add a home postcode in settings.

Caveats

  • Commute CO2e is an estimate using straight-line distance.
  • Employer signals reflect sector baselines, not company footprints.
  • Data gaps happen with missing locations, agencies, or unclear employers.

Sources

  • UK GHG conversion factors
  • Postcodes.io
  • Companies House API
  • ONS sector intensity baselines
  • SBTi Companies Taking Action snapshot
