# Data attribution

## Sources
- OS Open Names (OGL) — UK place index used for location matching.
- ONS Environmental Accounts (OGL) — sector emissions intensity baseline (SIC 2007).
- Companies House API — employer identity and SIC codes.
- SBTi Companies Taking Action (snapshot) — SBTi commitments/targets dataset.

## Notes
- Sector intensity values are industry baselines, not company-specific footprints.
